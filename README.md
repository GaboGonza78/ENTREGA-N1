# 🚀 Entrega N°1 - Backend

## 📌 Descripción
Proyecto backend desarrollado con Node.js y Express que implementa un módulo de mocking para generar usuarios y mascotas, junto con endpoints para testeo e inserción en base de datos MongoDB.

---

## ⚙️ Tecnologías utilizadas
- Node.js
- Express
- MongoDB
- Mongoose
- Faker
- Bcrypt
- Dotenv
